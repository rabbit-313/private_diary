### ログイン
1. [リンク](https://privatediary-by-ru.com/)を踏む
2. メールアドレス・パスワードを登録
3. 登録したメールアドレスから認証を行う

### 使い方
- 自分で日記を書きたい場合：「自分で新規作成」ボタンから日記作成
- AIに自動で書かせたい場合：「AIで新規作成」ボタンから日記作成
